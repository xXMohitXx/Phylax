# Quickstart Example

The fastest way to understand Phylax.

## Features Showcased

- `@trace`: Automatically tracking LLM executions.
- `@expect`: Defining deterministic boundaries.
- `execution()`: Grouping workflows contextually.

## Usage

Run the following from the root of this directory:

```bash
# Ensure phylax is installed
pip install phylax

# Run the quickstart
python app.py
```
