# AI Agent Template

A highly robust, production-ready template for a multi-step agent built with Phylax Multi-Agent Enforcement.

## Includes

- Pre-configured `.github/workflows/ci.yml` for CI enforcement.
- Agent step validation and tool sequence verification using `phylax.agents`.
- Baseline dataset contract testing ready out of the box (`dataset.yaml`).

## Quickstart

```bash
pip install phylax
python app.py
```
