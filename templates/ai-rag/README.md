# AI RAG Template

A highly robust, production-ready template for a Retrieval-Augmented Generation (RAG) agent featuring RAG-specific enforcement rules.

## Includes

- Pre-configured `.github/workflows/ci.yml` for continuous testing.
- RAG-enforcement rule mappings using `phylax.rag` (e.g., `ContextUsedRule`, `CitationRequiredRule`).
- Dataset contract testing baseline (`dataset.yaml`).

## Quickstart

```bash
pip install phylax
python app.py
```
