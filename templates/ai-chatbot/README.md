# AI Chatbot Template

A highly robust, production-ready template for a conversational AI chatbot.

## Includes

- Pre-configured `.github/workflows/ci.yml` for continuous testing.
- Built-in `phylax.guardrails` protection.
- Baseline dataset contract testing ready out of the box (`dataset.yaml`).

## Quickstart

```bash
pip install phylax
python app.py
```
